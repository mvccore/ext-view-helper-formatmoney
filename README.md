Format money by `Intl` extension or by locale formating conventions or by explicit or default arguments.

## Installation
```shell
composer require mvccore/ext-view-helper-formatmoney
```

## Example
```php
<b><?php echo $this->FormatMoney(123456.789); ?></b>
```
```html
<b>$ 123,456.789</b>
```
